## [1.0.0] - 2026-05-17

### Added
- **P4 Science Explorer**: Complete Singapore Primary 4 Science learning application
  - 9 comprehensive science topics: Living and Non-Living Things, Basic Needs, The Five Senses, Healthy Diet, Living Things & Habitats, Decomposers, Adaptations, Water Cycle, Weather & Climate
  - Interactive topic selection interface with card-based layout
  - Self-assessment exercises with immediate feedback
  - Audio pronunciation guide for scientific vocabulary
  - Progress tracking visualization
  - PDF export functionality for learning materials
  - Responsive design for desktop and tablet devices
  - Mallow design system implementation: Nunito typography, Caveat headings, cream/peach/mint palette, neo-brutalist style with ink borders
  
- **Vercel Deployment Configuration**:
  - vercel.json with optimized build and environment settings
  - .vercelignore for efficient deployment
  - Comprehensive README with deployment instructions
  - Static HTML hosting for optimal performance

### Technical Details
- Single-file self-contained HTML application (55KB)
- Pure CSS styling with modern gradients and shadows
- Vanilla JavaScript for interactivity and PDF generation
- No external dependencies or runtime requirements
- Accessible color contrasts and responsive breakpoints

### Deployment
- Deployed to Vercel static hosting
- Live at: p4wascience.vercel.app
- Zero cold-start latency, global CDN distribution
